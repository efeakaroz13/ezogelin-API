Documentation

This is the official documentation for this private fileserver. If you struggle with the instructions contact the developer.

State, city, neighbourhood API

API Endpoint:/api GET

Arguments:

  • key
  • city(if you don't specify it, it will give a list of cities that you can pass in to this variable)
  • district(You will enter one of the outputs of city and this will return you the neighbourhoods.)
  • nh(Enter one of the results from district and this will output the streets.)

Note: You need to specify every argument if you specify new ones. For example, if you used city and you want to get district you need to use both of them next time.
